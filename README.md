# 🔐 Password Generator & Copy Tool

A simple and responsive web application that generates secure, random passwords and lets you copy them with a single click.

## 🚀 Features

- ✅ Generates strong random passwords
- ✅ Includes uppercase, lowercase, numbers, and special characters
- ✅ Fully responsive design
- ✅ Lightweight and fast

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript

## 📂 How to Use

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Click the **Generate Password** button.
4. Click the **Copy icon** to copy the password to your clipboard.
